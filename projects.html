DParadox – Live Learning Marketplace Platform

Jan 2024 – Jun 2024

  • Built a production-grade live learning platform using Spring Boot and Angular, supporting tutor discovery, scheduling, live classes, chat, and recordings.
  • Implemented JWT-based authentication and role-based access control using Spring Security for students, tutors, and admins.
  • Designed a conflict-free scheduling system using interval overlap detection to prevent overbooking under concurrent requests.
  • Developed real-time communication using WebRTC and WebSockets, and built a scalable backend with PostgreSQL, AWS S3, Docker, and Kafka-based notifications.

RiskLens – Credit Risk & Fraud Detection System

Jul 2023 – Dec 2023

  • Built end-to-end machine learning models for credit risk scoring and transaction fraud detection using real-world financial datasets.
  • Performed data preprocessing and feature engineering, and trained models including Logistic Regression, Random Forest, and XGBoost.
  • Evaluated model performance using ROC-AUC, precision-recall, and false-positive rates.
  • Implemented explainable AI (SHAP) to provide feature-level insights and ensure model transparency.
  • Deployed models as REST APIs using FastAPI for real-time prediction services.
